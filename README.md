# mainsite-leyu-lab

本仓库用于归档和发布多个独立的 HTML 页面。这些页面通常用于展示、测试或作为静态资源使用，不针对特定域名或具体网站。

---

## 项目简介

mainsite-leyu-lab 是一个静态页面集合仓库。仓库中的每个 HTML 文件均为独立页面，彼此之间没有依赖关系。你可以直接通过浏览器打开这些文件，或将其部署到任意静态托管服务上进行访问。

该仓库的主要目的是集中管理和版本控制这些页面，便于后续修改、更新和历史回溯。

---

## 目录说明

仓库根目录下直接存放各个 HTML 文件及其相关资源（如 CSS、JavaScript、图片等）。目录结构如下：

```
/
├── index.html          # 仓库首页（可选）
├── page-example.html   # 示例页面
├── assets/             # 公共资源文件夹（可选）
│   ├── style.css
│   └── script.js
├── project-a/          # 项目 A 相关页面（可选）
│   ├── index.html
│   └── ...
└── project-b/          # 项目 B 相关页面（可选）
    ├── index.html
    └── ...
```

> 实际目录结构可能随页面增加或调整而变化。

---

## 页面归档说明

- 每个 HTML 页面均为独立完整页面，包含必要的样式和脚本引用。
- 页面之间无交叉链接，除非特别说明。
- 部分页面可能依赖外部资源（如字体库、图标库等），请确保网络连接正常。
- 归档的页面可能包含实验性功能或未完成内容，仅供参考或测试使用。
